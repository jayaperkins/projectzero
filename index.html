Introduction

At the beginning of the semester, I really did not know what to expect from this class. The name, "Web Technologies" seemed like a broad subject. It turns out that it is extremely broad! I never realized that so many tools/techniques were available to developers. After taking this class, I feel that I have a much deeper understanding of development and how to discover custom tools to aid my workflow.

Chrome's development tools are an indispensable asset. Sublime Text 2 is an affordable and understandable code editor. It also provides devlopers with many essential features. IP addresses enable communication between devices and the web. Ecommerce is an effective method for promoting a product. The flexibility of the web allows a seller to present their product to the public in any way that they wish.

Web technologies affect every individual. Whether you are a web user or web developer, a basic understanding of this tech can help you to personally become more efficient in your work. The question is: Will you take advantage of the resources that are readily available? I hope that this Project 0 can effectively show just a few of the amazing topics that I learned this semester.
