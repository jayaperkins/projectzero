Emmet

Ability to expand abbreviations into full HTML code

ul#nav>li.item$*4>a{Item $}

# or . to add an Id or Class to an element

Press "Tab" to expand

Click HERE to see before and after images

After watching the lynda.com tutorial, I had to try this package out in Sublime. Emmet works like a dream! From now on, I will make sure to utilize this powerful tool every time I create an HTML page! It is very effective and understandable. All of the commands that you type make sense and are readable.
