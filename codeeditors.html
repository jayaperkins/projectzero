This editor is powerful. This editor is customizable. This editor is simple.

Sublime Text 2 is a great software choice for any developer. Personally, I use WebStorm as my code editor. I really like the simplicity of the program. However, WebStorm is subscription based. The only reason that I use it is because it is free for students. I will most likely switch to a different program once WebStorm begins to charge students or I graduate. Sublime Text 2 would be one of my top choices when I change. The one-time purchase and flexibility of the software are appealing.

From multiple packages to project management to customization, Sublime Text 2 provides the user with some of the most cutting edge technologies and features available to coders. It is simplistic enough for beginners, but advanced enough for experts.

New functionality is constantly offered to increase the power of this software.

The question remains: Will you take advantage of it?
